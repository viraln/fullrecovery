## Recursive Creation Tutorial: A Practical Guide

Let's create a simple recursive function in Python to calculate the factorial:

```python
def factorial(n):
  if n == 0:  # Base case
    return 1
  else:
    return n * factorial(n-1)

print(factorial(5)) # Output: 120
```
